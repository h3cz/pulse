# Security Policy

Pulse is a browser-only focus and sleep audio tool. It is intended for personal, educational, and wellness-adjacent use only.

## Intended Use

- Local browser use
- Web Audio learning
- Personal focus/sleep audio experiments
- Authorized forks and self-hosted deployments

## Not Intended For

- Medical diagnosis, treatment, or therapy claims
- Hidden audio playback
- Tracking users without consent
- Collecting uploaded audio files

## Privacy Notes

Audio files selected in Pulse stay in the browser and are played through the local Web Audio graph. They are not uploaded by the app.

The public build ships with no remote analytics. Forks that add analytics should document the endpoint, collected fields, retention window, and opt-out behavior.

## Reporting Issues

If you find a security or privacy issue, open a private security advisory or contact the maintainer directly.
